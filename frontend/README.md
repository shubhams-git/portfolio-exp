# Frontend

React + TypeScript portfolio frontend built with Vite.

This package contains the recruiter-facing UI for "The Layered Matrix" portfolio, including:

- homepage sections
- project quick preview modal
- dedicated project case-study routes
- shared navigation and footer
- contact form UI that posts to the backend API

## Run

```bash
npm install
npm run dev
```

## Verify

```bash
npm run build
npm run lint
```

## Key Files

- `src/App.tsx`
- `src/App.css`
- `src/index.css`
- `src/pages/HomePage.tsx`
- `src/pages/ProjectPage.tsx`
- `src/content/portfolio-content.ts`
